# Stage 3: Data Curation and Pre-Processing

## Overview
This stage focuses on curating, validating, and preparing candidate NirK and PCuAC homologues identified from the BLASTp searches in Stage 2. The aim is to construct a reliable dataset by confirming biological relevance, removing low-quality sequences, and generating accurate species-level labels for downstream feature extraction and machine learning.

## Workflow
1. Validating candidate homologues using conserved residue checks
2. Applying sequence quality filters, removing duplicate sequences, selecting representative accessions
3. Constructing the final labelled dataset
4. NirK and PCuAC sequences are evaluated to ensure that only high-confidence homologues are included before proceeding to feature extraction and predictive modelling.
